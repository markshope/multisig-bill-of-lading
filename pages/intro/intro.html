Multisig Blockchain Bill of Lading Example

This toolkit has been built with educational purposes in mind!
It is meant to illustrate how multisig technology can be used in the supply chain setting.
You must be extremely careful when using real/live/mainnet data/keys/credentials!
A web browser is not a safe environment to either create strong cryptographic keys and/or paste sensitive information.
